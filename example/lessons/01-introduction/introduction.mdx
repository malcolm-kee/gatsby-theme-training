## About this site

There are two types of code examples:

1. The code examples that you can edit in this site directly. The edit icon at corner indicates you can edit them.

```js live fileName=some.js
function greet(message) {
  return 'Hi! ' + message;
}

console.log(greet('I am Malcolm!'));
```

2. The code examples that you can only view. The eye icon at corner indicates you can only view them.

```js
function greet(message) {
  return 'Hi! ' + message; // highlight-line
}

console.log(greet('I am Malcolm!'));
```
